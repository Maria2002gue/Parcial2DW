Proyectos Realizados

SIEL(Sistema de informacion egresados)

Este proyecto consistió en la creación de una plataforma web que permite a los egresados de la universidad La Salle registrar y actualizar sus datos, facilitando el seguimiento de su trayectoria profesional y académica. Mi rol principal fue el desarrollo de la interfaz del software, asegurando que fuera intuitiva, accesible y visualmente atractiva.

Tecnologías: HTML, CSS, JavaScript

Encriptador

En este proyecto, desarrollé un encriptador de texto que permite cifrar y descifrar mensajes utilizando un algoritmo personalizado. Este encriptador fue diseñado para ofrecer un nivel básico de seguridad al transmitir información confidencial.

Tecnologías: HTML, CSS, JavaScript

MyFirstGame

Este proyecto fue mi primera experiencia en desarrollo de videojuegos y se llevó a cabo en C# con Visual Studio y MGCB (MonoGame Content Builder). El juego incorpora gráficos y mecánicas básicas de interacción, explorando conceptos como detección de colisiones, control de eventos en tiempo real y lógica de movimiento.

Tecnologías: C#, Visual Studio y MGCB (para la gestión de contenido gráfico y multimedia).
